# Jam Game - Carnival of Masks

## Overview
A third-person carnival game featuring animal characters wearing magical masks with unique abilities. Players navigate a darkly-themed carnival filled with twisted challenges and surprises.

## Game Concept
- **Setting**: Sinister carnival/fair environment
- **Characters**: Anthropomorphic animals
- **Core Mechanic**: Complete carnival mini-games to progress

## Mini-Games
- **Dunk Tank**: Knock characters into water
- **Balloon Pop**: Burst balloons under time pressure
- **Strength Meter**: Test your power with a hammer challenge
- **Bowling**: Knock down pins into a goal
- *Hidden surprises and dark twists throughout*

## Mask Abilities
Each character's mask grants special powers:
- **Buff Abilities**: Enhanced strength, speed, or defense
- **Bug/Quirk Abilities**: Unusual mechanics or glitches for unique gameplay

## Features
- Darkly atmospheric carnival setting
- Unpredictable events and surprises
- Character-specific mask powers affecting gameplay
- Multiple mini-game challenges
